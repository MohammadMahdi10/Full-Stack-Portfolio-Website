My Projects

Here are some of the projects I've worked on during my studies and personal development. They showcase my skills in programming, problem-solving, and teamwork.

Dungeon Crawler Game

Description: This is my first-ever game project made using Godot. It is a dungeon crawler game where the player navigates through a dungeon, avoiding obstacles and enemies. I implemented A* pathfinding to allow for intelligent movement within the dungeon.

Technologies Used: Godot, GDScript, A* Pathfinding, 2D Arrays, Object-Oriented Programming (OOP)

Key Features:

  • A* Pathfinding for enemies and player movement
  • Dynamic dungeon generation
  • Collision detection and object interaction
  • Use of 2D arrays to manage dungeon layout
Hack and Slash Game

Description: This is a hack-and-slash game I developed using Python and the PyGame library. The game allows players to battle through different levels, encountering enemies, collecting loot, and using a variety of combat techniques. The project lasted over four months, making it the longest project I've worked on so far.

Technologies Used: Python, PyGame, Object-Oriented Programming (OOP), 2D Arrays, Game Design

Key Features:

  • Character movement and combat mechanics using object-oriented principles
  • Level design and generation with 2D arrays
  • Custom graphical design for game elements (sprites, backgrounds, UI)
  • Multiple enemy types with varying behavior and difficulty
Text-Based Train Game

Description: In this team project, we developed a text-based train game where players manage a train station, schedule trains, and navigate through various challenges. The core logic of the game was based on linked lists to manage the train schedule and track train movement through different stations.

Technologies Used: Python, Linked Lists, Team Collaboration

Key Features:

  • Implementation of linked lists to track train schedules and stations
  • Text-based user interface (CLI) for game interaction
  • Train movement logic and scheduling
  • Team collaboration and version control
